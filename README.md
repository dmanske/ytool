# YTool - Oficial

Ferramenta completa para download de videos do YouTube e Instagram.

## Funcionalidades

- Download de videos YouTube e Instagram
- Transferencia de inscricoes e playlists entre contas
- Fila de downloads com player embutido
- Corte de trecho de video
- Historico persistente
- Tema claro/escuro
- Interface web local em PT-BR

## Stack

- Python (FastAPI backend)
- Swift (app macOS nativo)
- yt-dlp (engine de download)

## Como rodar

python app.py
