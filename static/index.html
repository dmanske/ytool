    Google OAuth
    
          
    
            Necessário para Inscrições e Playlists. Crie as credenciais em
            console.cloud.google.com → APIs & Services → Credentials → OAuth 2.0 Client IDs.
            Adicione http://localhost:8000/api/subscriptions/oauth/callback como URI de redirecionamento.
            Defina GOOGLE_CLIENT_ID e GOOGLE_CLIENT_SECRET no arquivo .env.

          Como usar o YTool
        
    
        
      
    

      
    

        
        
    
          
    
            
          
    
          
    
            
    Downloads
    
            
    Baixe vídeos e áudios do YouTube e Instagram (apenas conteúdo público).
    
            
      
              
    1. Cole a URL do vídeo no campo de URL — o app detecta automaticamente se há uma URL do YouTube ou Instagram no seu clipboard.
      2. 
              
    2. Clique em Inspecionar para ver os formatos disponíveis e uma prévia do vídeo (título, duração, canal).
      3. 
              
    3. Escolha a qualidade, formato e categoria desejados.
      4. 
              
    4. Marque Somente áudio para baixar apenas o áudio em mp3/m4a.
      5. 
              
    5. Marque Legendas e informe os idiomas (ex: en,pt) para baixar legendas junto.
      6. 
              
    6. Use Nome do arquivo para definir um nome personalizado (opcional).
      7. 
              
    7. Marque Cortar trecho para baixar apenas uma parte do vídeo (ex: de 1:30 a 3:00).
      8. 
              
    8. Clique em Baixar para download direto, ou Adicionar à fila para baixar várias URLs em sequência.
      9. 
              
    9. Durante o download, use o botão ✕ ao lado da porcentagem para cancelar.
      10. 
            
    
            
    
              
              Você pode arrastar um link do navegador direto pro campo de URL. O histórico é salvo automaticamente e persiste entre sessões.
            
    
          
    
        
    

        
    

        
        
    
          
    
            
          
    
          
    
            
    Inscrições
    
            
    Exporte, importe ou transfira suas inscrições (canais seguidos) entre contas do YouTube.
    
            
      
              
    1. Conecte a Conta de Origem (de onde vêm as inscrições) clicando em "Conectar com Google".
      2. 
              
    2. Conecte a Conta de Destino (para onde vão as inscrições) da mesma forma.
      3. 
              
    3. Use Exportar → Arquivo para salvar as inscrições da conta de origem em JSON.
      4. 
              
    4. Use Importar Arquivo → para carregar um JSON/CSV e inscrever a conta de destino nos canais.
      5. 
              
    5. Use Transferir A → B para copiar diretamente da origem para o destino sem arquivo intermediário.
      6. 
            
    
            
    
              
              A transferência respeita o limite da API do YouTube (~1 inscrição por segundo). Para 500 canais, leva cerca de 8 minutos.
            
    
          
    
        
    

        
    

        
        
    
          
    
            
          
    
          
    
            
    Playlists
    
            
    Copie playlists (públicas e privadas) de uma conta do YouTube para outra.
    
            
      
              
    1. Certifique-se de que as contas de Origem e Destino estão conectadas (faça isso na aba Inscrições).
      2. 
              
    2. Clique em Carregar Playlists da Origem para listar todas as playlists da conta de origem.
      3. 
              
    3. Selecione as playlists que deseja copiar usando os checkboxes. Use "Selecionar todas" para marcar tudo.
      4. 
              
    4. Clique em Transferir Selecionadas → Destino e acompanhe o progresso.
      5. 
            
    
            
    
              
              Playlists privadas são recriadas como privadas na conta de destino. Playlists públicas são recriadas como públicas.
            
    
          
    
        
    

        
    

        
        
    
          
    
            
          
    
          
    
            
    Configurações
    
            
    Personalize o comportamento do app.
    
            
      
              
    • Pasta de downloads: onde os arquivos são salvos. Padrão: ~/Downloads/YTool.
      • 
              
    • Categorias: organize seus downloads em pastas por categoria (ex: Música, Tutoriais).
      • 
              
    • Google OAuth: credenciais necessárias para usar Inscrições e Playlists. Crie em console.cloud.google.com e adicione ao arquivo .env.
      • 
            
    
          
    
        
    

        
    

        
        
    
          
    
            
          
    
          
    
            
    Limitações conhecidas
    
            
      
              
    • Instagram: apenas conteúdo público (posts, reels, stories de contas públicas). Contas privadas não são suportadas.
      • 
              
    • YouTube: downloads de vídeos com DRM ou restrição de idade podem falhar.
      • 
              
    • Playlists: vídeos removidos ou privados na conta de origem não são copiados.
